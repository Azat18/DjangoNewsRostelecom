Рабочий проект находится в приложении Rostelecom_study/NewsStudyRostelecom/magazine
Для перехода на страницу со списком новостей нужно кликнуть НОВОСТИ в горизонтальном меню
Для перехода на страницу первой новости нужно со страницы НОВОСТИ перейти по кнопке ПОДРОБНЕЕ в первой новости сверху
Для перехода на страницу аккаунта пользователя нужно перейти в верхнем меню по ссылке Аккаунт
Для возврата на главную страницу нужно перейти в верхнем меню по ссылке Главная
